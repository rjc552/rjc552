Drugs in the Body

My final project idea allows you to see where drugs are active in the body. The user can search a drug such as "Ibuprofen", and the website will show (and tell) you which body tissues this drug targets.

Genes

I want to create a database of 15-30 drugs of the most well known drugs, identify which genes they target, and then make a body map of where those genes occur. If a drug targets a specific protein in a specific gene, the body tissues that contain the gene will be colored in. The darker the color the higher the confidence. The website will tell you the names of the top hitting tissues, and the genes it is hitting. you can see this information by hovering the mouse over the body parts.

Tools+Credit

I will be using JQuery and Bootstrap. I will also use various free use websites that contain data on genes, drugs and tissues. My project is directly inspired by Tissues.jensenlab.org provides data on genes and tissues, so i would like to have a loading screen crediting that website, and a button that links to it.

Design and User interface

I want the design to be as pretty and clean as possible. Lots of science websites are a bit ugly, so it would be nice to create a science project type-thing that looks nice. Furthermore, I may include, at the bottom, a list of the drugs. That way, user can click on the drugs instead of searching the drug.
